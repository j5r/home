Inserir $\LaTeX$ em arquivo HTML (site)


Para inserir texto $\LaTeX$ em um HTML, precisa-se adicionar alguns scripts na tag <head>.


<script type="text/x-mathjax-config">
MathJax.Hub.Config({tex2jax: {inlineMath: [['$','$'], ['\\(','\\)']]}});
</script>
<script type="text/javascript"
src="https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.7/MathJax.js?config=TeX-AMS-MML_HTMLorMML">
</script>
Clique aqui para copiar.

Agora que você carregou os scripts para o seu documento HTML, você pode escrever fórmulas $\LaTeX$ usando algo como


$$x = \dfrac{-b \pm \sqrt{b^2 - 4\cdot a \cdot c}}{2a}$$ Clique aqui para copiar.

Isso produzirá o seguinte resultado:


$$x = \dfrac{-b \pm \sqrt{b^2 - 4\cdot a \cdot c}}{2a}$$
